Durant l'année de terminale, nous devions réaliser un site Web ayant pour sujet une notion abordée au cours de l'année. Il devait contenir des références historiques ainsi qu'une partie technique concernant le sujet. Pourquoi ce projet ?

Le "Gai Séjour" est un meublé saisonnier situé aux Sables-d'Olonne. Ce projet est un site Web vitrine permettant de promouvoir celui-ci. Il est en cours de développement depuis juin 2022. Les utilisateurs de ce site pourront avoir la possibilité de consulter un calendrier afin de réserver leur séjour. De plus, en consultant ce site vous pourrez découvrir le meublé ainsi que la ville des Sables-d'Olonne.

Technologies utilisées

Date de création

Paradigme de Programmation

Pourquoi ce projet ?

Découvrez le jeu de la bataille automatisé ! Codé en Python et utilisant la Programmation Orientée Objet, vous pourrez affronter vos amis ou un bot grâce à ce dernier. Il regroupe plusieurs classes permettant de construire les cartes, le jeu de carte ainsi que le joueur. A chaque tour, vous pourrez apprécier une petite animation où vous retrouverez votre pseudo ainsi que le nombre de cartes que en votre possession dans votre main. A la fin de la partie, un message s'affiche avec le nom du gagnant.
